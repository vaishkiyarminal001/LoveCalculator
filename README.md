# Love Calculator Web App 💖

The Love Calculator Web App is a fun and interactive web application that allows you to calculate the love percentage between two individuals. 😍💑

<img align="right" alt="Coding" width="100%" src="/1.png"><br>

## Features ✨

🌟 User-friendly interface with romantic aesthetics.
🌟 Input fields for entering names.
🌟 Dynamic calculation of love percentage.
🌟 Result messages ranging from "Poor Relationship" to "Made for each other."
🌟 Soothing background music for a romantic ambiance.
🌟 Input validation to ensure names are at least 3 characters long.

## Technologies Used 🛠️

- HTML 📄
- CSS 🎨
- JavaScript 🚀
- HTML5 Audio Element 🎵

## Usage 📝

1. Enter the names of two individuals in the input fields.
2. Click the "Calculate" button to see the love percentage and a sweet message.

## Target Audience 👫

The Love Calculator Web App is perfect for couples, friends, or anyone looking to have a bit of fun while assessing love compatibility. 😄
